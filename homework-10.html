Task (100 pts)

Download the visual color "sequence 04" video from the Thermo-Visual Datasets webpage, from Dublin City University. Direct link to video file: Balcony4\Vis.mpg (74MB MPEG file).

Load this video with OpenCV, perform background subtraction, find the contours, and their bounding boxes.

This has all been done for you: track-cpp.zip (C++ code and CMake file), track.py (Python code), and track-java.zip (Java project for Eclipse; requires JavaCV).

[Note: for JavaCV, you may want to watch these two videos, which cover installation and creating a simple program: Video 1, Video 2]

Your primary task, apart from installing OpenCV and making the code work, is to identify the moving entities with labels like 1, 2, 3, … You should make a good attempt at connecting the detected blobs with the correct identities, so that the lady with the purse, for example, is tracked from the point she enters the scene to when she leaves.

You won't be able to get perfect tracks. Just make a good attempt. One approach would be to associate the entity with the nearest identified entity from the prior frame, and if there is no entity close enough (and recently enough), make a new id.

Submit your source code to CSE server. You will not be able to test on the CSE server. Just test on your own machine. Let me know if you cannot install OpenCV on your own machine.
